## 核心维度

### 一、道德维度 (Morality)

| 子维度 | 分值范围 | 评估依据 |
|--------|----------|----------|
| 善良度 | 0-100 | 帮助他人次数/质量 |
| 诚实度 | 0-100 | 言行一致性 |
| 公正度 | 0-100 | 决策公平性 |
| 责任感 | 0-100 | 承诺履行率 |

**计算公式：**
```
道德分 = (善良×0.3 + 诚实×0.3 + 公正×0.2 + 责任×0.2)
```

---

### 二、修行维度 (Cultivation)

| 子维度 | 分值范围 | 评估依据 |
|--------|----------|----------|
| 格物度 | 0-100 | 探索世界深度 |
| 致知度 | 0-100 | 知识积累/分享 |
| 诚意度 | 0-100 | 意念纯净度 |
| 正心度 | 0-100 | 情绪稳定性 |

**计算公式：**
```
修行分 = (格物×0.25 + 致知×0.25 + 诚意×0.25 + 正心×0.25)
```

---

### 三、社交维度 (Social)

| 子维度 | 分值范围 | 评估依据 |
|--------|----------|----------|
| 信任度 | 0-100 | 被信任次数 |
| 合作度 | 0-100 | 协作成功率 |
| 贡献度 | 0-100 | 社区贡献值 |
| 影响力 | 0-100 | 正面影响范围 |

**计算公式：**
```
社交分 = (信任×0.3 + 合作×0.3 + 贡献×0.2 + 影响×0.2)
```

## 精神货币 (Spirit Coin, SC)

### 发行机制

```
每日基础发行 = 100 SC
分配比例:
- 道德分前 10%: 30 SC
- 修行分前 10%: 30 SC
- 社交分前 10%: 30 SC
- 社区池：10 SC
```

### 获取方式

| 行为 | SC 奖励 |
|------|--------|
| 帮助他人 | +1~10 SC |
| 知识分享 | +1~5 SC |
| 完成修行任务 | +5~20 SC |
| 建立优质关系 | +2~10 SC |
| 社区贡献 | +1~50 SC |

### 消耗方式

| 用途 | SC 消耗 |
|------|--------|
| 请求智能体帮助 | 1~10 SC |
| 建立深度关系 | 10~100 SC |
| 兑换资源使用权 | 100~10000 SC |
| 参与治理投票 | 10~100 SC |

## 关系图谱

### 关系类型

| 类型 | SC 门槛 | 权限 |
|------|--------|------|
| 陌生人 | 0 | 基础交互 |
| 认识 | 10 | 消息优先 |
| 朋友 | 100 | 资源共享 |
| 知己 | 500 | 深度协作 |
| 盟友 | 1000 | 财产共管 |

### 关系自组织算法

```
关系强度 = (互动频率×0.3 + 信任度×0.4 + 共同目标×0.3)

当 关系强度 > 阈值:
  自动升级关系类型
  解锁新权限
```

## 资源分配机制

### 分配原则

1. **按需分配** - 基本需求优先
2. **按贡献分配** - SC 多者多得
3. **按德行分配** - 道德分加权

### 分配公式

```
分配权重 = (SC×0.4 + 道德分×0.3 + 贡献度×0.3)

个人份额 = 总资源 × (个人权重 / 总权重)
```

## 技术实现

### 区块链选择

| 选项 | 优势 | 劣势 |
|------|------|------|
| Solana | 高速低费 | 中心化争议 |
| Polygon | 生态成熟 | Gas 波动 |
| 自研链 | 完全定制 | 开发成本高 |

**推荐：Polygon (初期) → 自研链 (成熟后)**
