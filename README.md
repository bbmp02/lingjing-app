# 灵境 (Spirit Realm) APP

> 精神货币与关系自组织系统

**状态：** 🚧 开发中  
**版本：** v0.1.0  
**创建时间：** 2026-03-06

---

## 🌟 愿景

构建一个基于精神价值量化的人类协作新架构，通过道德、修行、社交三维度评分，实现更公平的资源分配和关系自组织。

---

## 📋 核心功能

### 精神货币系统
- 道德维度 (善良/诚实/公正/责任)
- 修行维度 (格物/致知/诚意/正心)
- 社交维度 (信任/合作/贡献/影响力)

### 关系自组织
- 5 层关系：陌生人 → 认识 → 朋友 → 知己 → 盟友
- 自动升级，基于互动 + 信任 + 共同目标

### 资源分配
- 按贡献分配
- 按德行加权
- 透明可审计

---

## 🛠️ 技术栈

| 层级 | 技术 |
|------|------|
| 移动前端 | React Native + TypeScript |
| Web 前端 | Next.js 14 + TailwindCSS |
| 后端 | NestJS + TypeScript |
| 数据库 | PostgreSQL + Redis (Railway) |
| AI 服务 | OpenClaw + Llama 3 |
| 部署 | Vercel + Railway |
