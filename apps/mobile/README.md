# 灵境 APP - React Native 前端

> 精神货币与关系自组织系统 - 移动端

## 技术栈
- React Native (Expo)
- TypeScript
- Zustand (状态管理)
- React Navigation

## 模块
- 用户模块
- 精神货币模块
- 关系图谱模块
- 互动记录模块

## 开始开发
```bash
cd apps/mobile
npx create-expo-app@latest
npm install
npx expo start
```

## API 配置
```env
EXPO_PUBLIC_API_URL=https://lingjingai.chat/api
```
